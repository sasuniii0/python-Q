# Python Exercises Collection

This repository contains **28 Python exercises** covering **fundamentals, functions, lambda & functional programming, OOP, and file handling**. The exercises are categorized into separate Jupyter notebooks for better organization and learning.

## üìù Exercise Highlights

- **Fundamentals & Operations**:  
  - Duplicate check, absolute value, largest number, factorial, triangle calculations, weekdays/weekends, sum of numbers using loops, break/continue.

- **Functions**:  
  - Sum of squares, adding corresponding elements, total cost with discounts, average calculations, temperature conversions.

- **Lambda & Functional Programming**:  
  - Maximum of two numbers, squaring numbers, adding lists, filtering numbers divisible by 3 but not 5.

- **OOP**:  
  - Car and Circle classes with methods, StudentRecords class demonstrating strong encapsulation.

- **File Handling**:  
  - Reading files, writing only even-numbered lines to new files.
